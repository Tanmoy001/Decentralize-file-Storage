Hardhat provied a local blockchain evironment which runs on the cli
****For Hardhat installion*******
npm install --save-dev hardhat@2.12.4
npx hardhat
npm install --save-dev "hardhat@^2.12.4" "@nomicfoundation/hardhat-toolbox@^2.0.0"

***Install react app****
npm install create-react-app User

***run react app***
npm start

***run hardhat ***
npx hardhat node


**deploying smart contract **
npx hardhat run --network localhost scripts/deploy.js
